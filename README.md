# Oreo 的个人作品集

这是我的个人作品集网站，展示了我的项目经验和技能。

## 🚀 本地运行

```bash
npm install
npm run dev
```

访问 http://localhost:3000 查看效果

## 📦 部署到 Vercel

### 方法 1: 通过 Vercel CLI

```bash
npm install -g vercel
vercel
```

### 方法 2: 通过 GitHub

1. 把代码推到 GitHub
2. 在 Vercel 导入项目
3. 自动部署完成

## ✏️ 自定义内容

### 修改联系信息
编辑 `index.html`，找到 "联系我" 部分，修改：
- 闲鱼用户名
- 邮箱地址

### 修改作品链接
编辑 `index.html`，找到 "作品展示" 部分，更新你的项目链接。

### 修改价格
编辑 `index.html`，找到 "报价范围" 部分，调整你的价格区间。

## 🛠️ 技术栈

- Vite - 构建工具
- Tailwind CSS - 样式框架
- 原生 JavaScript - 无框架依赖
