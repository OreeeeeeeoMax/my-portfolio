# 子域名配置教程

## 目标

将以下 3 个项目都绑定到 `oreocolor.xyz` 域名，全部可在中国访问：

| 子域名 | 项目 | 用途 |
|--------|------|------|
| oreocolor.xyz | my-portfolio | 个人作品集（已配置）✅ |
| corp.oreocolor.xyz | corp-site-template | 企业官网作品集 |
| color.oreocolor.xyz | color-v2 | 色觉体验工具 |

---

## 步骤 1：在 Vercel 添加域名（10 分钟）

### 1.1 配置 corp.oreocolor.xyz

1. **登录 Vercel**
   - 访问 https://vercel.com
   - 进入 `corp-site-template` 项目

2. **添加域名**
   - 点击 `Settings` → `Domains`
   - 点击 `Add Domain`
   - 输入：`corp.oreocolor.xyz`
   - 点击 `Add`

3. **记录 DNS 信息**
   - Vercel 会显示 DNS 配置信息：
     ```
     Type: CNAME
     Name: corp
     Value: cname.vercel-dns.com
     ```
   - **复制这个信息，下一步要用**

### 1.2 配置 color.oreocolor.xyz

1. **进入 color-v2 项目**
   - 在 Vercel 切换到 `color-v2` 项目

2. **添加域名**
   - 同样步骤：`Settings` → `Domains` → `Add Domain`
   - 输入：`color.oreocolor.xyz`
   - 点击 `Add`

3. **记录 DNS 信息**
   - Vercel 会显示：
     ```
     Type: CNAME
     Name: color
     Value: cname.vercel-dns.com
     ```

---

## 步骤 2：在阿里云配置 DNS（5 分钟）

### 2.1 登录阿里云 DNS

1. **登录阿里云**
   - 访问 https://dns.console.aliyun.com
   - 找到域名 `oreocolor.xyz`
   - 点击 `解析`

### 2.2 添加 CNAME 记录

点击 `添加记录`，填写以下信息：

#### 记录 1：corp 子域名
| 字段 | 值 |
|------|-----|
| 记录类型 | CNAME |
| 主机记录 | corp |
| 记录值 | cname.vercel-dns.com |
| TTL | 10 分钟（默认） |

点击 `确认`

#### 记录 2：color 子域名
| 字段 | 值 |
|------|-----|
| 记录类型 | CNAME |
| 主机记录 | color |
| 记录值 | cname.vercel-dns.com |
| TTL | 10 分钟（默认） |

点击 `确认`

### 2.3 验证配置

配置完成后，DNS 记录应该是这样的：

| 主机记录 | 记录类型 | 记录值 | 状态 |
|----------|----------|--------|------|
| @ | A | 76.76.21.21 | 已有（主站）✅ |
| corp | CNAME | cname.vercel-dns.com | 新增 ✅ |
| color | CNAME | cname.vercel-dns.com | 新增 ✅ |

## 常见问题

### Q1: DNS 一直显示 Pending？
**A:** 正常现象，DNS 全球传播需要 10-30 分钟，最多 48 小时。喝杯咖啡等一等~

### Q2: Vercel 显示 `Invalid Configuration`？
**A:** 检查阿里云 DNS 配置：
- 记录类型必须是 `CNAME`
- 记录值必须是 `cname.vercel-dns.com`
- 不要用 `www` 前缀

### Q3: 配置后 404？
**A:** 可能原因：
1. DNS 还没生效（等 30 分钟）
2. Vercel 项目没有正确部署
3. 域名拼写错误（检查是否有空格）

### Q4: 可以用 www 子域名吗？
**A:** 可以，但不推荐：
- `www.corp.oreocolor.xyz` 太长
- 直接用 `corp.oreocolor.xyz` 更简洁

### Q5: 要付费吗？
**A:** 不需要！
- Vercel 免费版支持绑定自定义域名
- 阿里云 DNS 解析免费
- 子域名免费创建，无限制
